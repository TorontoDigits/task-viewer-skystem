# 🚀 Task Viewer System

A modern, feature-rich task management dashboard built with **HTML, CSS, and TypeScript**.

---

## ✨ Features

- Create, edit, and delete tasks
- Status tracking (Pending, In Progress, Completed, Overdue)
- Priority levels
- Advanced filtering
- Bulk actions
- File uploads
- Export (CSV, JSON, XML, Excel, PDF)
- Pagination support

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- TypeScript

---

## ▶️ How to Run

1. Clone the repository
2. Open `index.html` in your browser

No backend required.
